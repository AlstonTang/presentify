# Presentify
A web-based application to convert markdown into a presentation

## Features
- Uses [reveal.js](https://github.com/hakimel/reveal.js)
- Supports LaTeX syntax via [KaTeX](https://github.com/KaTeX/KaTeX).
- Supports Mermaid diagrams.

## Resizing
- **Images**: `![Alt](url =WxH)` or `![Alt](url =W)`.
- **Mermaid**: Add `%% size: 50%` as the first line of the code block.

## Fun Fact
You can *literally* turn this very *document* into a Presentify presentation!
- Get started by cloning this repo and running `npm run dev`
